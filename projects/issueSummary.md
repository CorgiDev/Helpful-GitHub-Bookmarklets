# Issue Summary Script

This script allows you to provide a url of a project board. Then it will provide you with a list of stats related to the issues, including:

- total number of issues
- a breakdown by severity level (Severity-1, Severity-2, and Severity-3)
- A breakdown by WCAG label
- A list of issues that specifically mention screen readers

These stats are followed by a general summary generated by Copilot about the issues.

## Required Installed Extensions or Modules

The following are modules and extensions required to run this script:

- **[Node.js](https://nodejs.org/) 18+** — Runtime required to execute the script (uses built-in modules: `fs`, `path`, `readline`, `child_process`)
  - Install by going to the Node.js site.
  - Install version 18+.
- **[GitHub CLI (`gh`)](https://cli.github.com/)** — Used to query the GitHub GraphQL API with your existing authenticated session; must be signed in (`gh auth login`)
  - Install with `brew install gh`
- **[GitHub Copilot CLI (`copilot`)](https://githubnext.com/projects/copilot-cli)** — Used to generate the General Summary section; must be authenticated and available on your PATH
  - Install with `brew install copilot-cli`
